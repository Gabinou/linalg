# linalg (Linear Algebra) 

## Installation
Add ```linalg.c``` and ```linalg.h``` to your source code.

## Features
```linalg.h``` contains ```darr.h```.

## Motivation
Make a very simple linear algebra library, with simple operations. For 3D and 2D matrices.
